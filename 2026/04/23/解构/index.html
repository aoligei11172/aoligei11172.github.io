解构

2026/4/23 JavaScript

解构是ES6引入的语法糖，可以从数组或对象中提取值并赋给变量，使代码更简洁、可读性更高，是一种优雅的语法。

解构变量如果是新声明的变量则不能和已有变量冲突。

数组解构

将数组元素快速批量赋给一系列变量。

例：

const arr = [100,60,80];
const [max,min,avg] = arr;

变量数大于单元值数，多余的变量赋为undefined，

变量数小于单元值数，多余的单元值不会被赋给变量，可用剩余变量获取

防止undefined，可设置默认值：

const [a = 0,b = 0] = [1,2]

根据需求可使用空格跳过不需要的单元值：

const [a,b, ,d] = [1,2,3,4]

对象解构

将对象属性和方法快速批量赋值给一系列变量

把对象里的属性取出来放到变量里，左边的变量名必须和右边的属性名一模一样

const {name,age} = {name: 'd4c',age: 18}

对象解构可以从一个对象中提取变量并同时修改新的变量名

什么值: 赋给谁

const obj = {
    uname: 'A',
    age: 18    
    };
const {uname: name } = obj;
console.log(name); //输出"A"

数组对象解构

const [{name,age}] = [{name: 'd4c',age: 18}]

多级对象解构

const obj1 = {
	a: 'A'
    obj2:{
		b:'B'
}
}
const {a , obj2:{b}} = obj1
