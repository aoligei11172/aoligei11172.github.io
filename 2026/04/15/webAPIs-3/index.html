DOM事件监听

2026/4/15 JavaScript
事件监听
元素对象.addEventListener('事件类型',function);
//语法

事件对象

是事件绑定的回调函数的第一个参数，通常命名为event，内涵事件触发时的相关信息

常用属性

type 获取事件类型

clientX/clientY 获取光标相对窗口左上角的坐标

offsetX/offsetY 获取光标相对DOM元素左上角的位置

key 按下键的值

常用事件

1.鼠标事件

click 单机

dbclick 双击

mousedown 鼠标任意键

mouseup 鼠标松开

mouseenter(无冒泡效果) mouseover(有冒泡效果) 鼠标进入

mouseleave(无冒泡效果) mouseout(有冒泡效果) 鼠标离开

2.键盘事件

keydown 按下任意键

keyup 松开按键

事件监听不能直接监听是否按下某一个键，可以使用

document.addEventListener('keydown',(event)=>{
    if(event.key==='某一按键'){}
});

来设置按下某些按键的情景

3.表单事件

input <input>/<textarea> 值改变（实时）

focus 聚焦

blur 失焦

submit 提交

change 失去焦点且值改变时（<select> 立即触发）

4.窗口事件

load 加载完成

scroll 滚动滚动条

5.M端事件

touchstart 手指触摸到一个DOM元素

touchmove 手指在一个DOM元素上滑动

touchend 手指从DOM元素上离开

事件监听版本

1.事件源.on事件 = function(){}

2.事件源.addEventListener(‘事件’,function);

on方式会被覆盖,addEventListener可以有更多特性
